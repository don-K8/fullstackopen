# Exercise 0.6: New note in Single page app diagram
```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User types a note and clicks Save

    Note right of browser: spa.js intercepts the click

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa (JSON payload: { "content": "new note", "date": "2026-03-28" })
    activate server
    server-->>browser: 201 Created { "message": "note created" }
    deactivate server

    Note right of browser: JS updates the page itself without any redirects or reloads
```
